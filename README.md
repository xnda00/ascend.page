# 🚀 Ascend.page
Minimalist blog built with **Astro**. Documenting my journey through graphics programming, AI, and related experiments.

### 🛠️ Tech Stack
* **[Astro](https://astro.build/)** - Framework
* **[Figma](https://www.figma.com/)** - Prototyping
* **[CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)** - Styling
* **[Markdown](https://www.markdownguide.org/)** - Blog articles

### 📂 Main Structures
* `/src/pages/posts` – Markdown articles
* `/src/components` – Modular UI elements
* `/src/styles` – Global CSS & variables
